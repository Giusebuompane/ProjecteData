Configuración inicial: git config --global user.name "Tu Nombre"
git config --global user.email "tu@email.com"
Clonar un repositorio remoto: git clone https://github.com/usuario/repositorio.git
git init

Ver estado de los archivos:git status
Ver los cambios realizados:git diff

Añadir archivos al área de preparación:git add nombre_del_archivo
 o todos: git add .
Hacer un commit (guardar cambio):git commit -m "Mensaje explicando el cambio"
  Conexión con repositorio remoto
Ver repositorio remoto:

git remote -v
Añadir repositorio remoto (si no existe):

git remote add origin https://github.com/usuario/repositorio.git
Enviar cambios al remoto:git push origin main
# o master si es la rama principal
Traer cambios del remoto:

git pull origin main
Ramas
Crear una nueva rama:

git branch nueva-rama
Cambiar de rama:
git checkout nombre-rama
Crear y cambiar en un paso:
git checkout -b nueva-rama
Fusionar ramas:
git merge nombre-rama
